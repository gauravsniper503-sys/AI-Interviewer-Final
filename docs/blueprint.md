# **App Name**: InterviewAce

## Core Features:

- Interview Simulation: Simulate interviews for various roles using AI.
- Interview Type Selection: Allow users to select from pre-defined interview types (e.g., Software Engineer) or create custom interview types.
- Question Generation: Generate contextually appropriate interview questions based on the selected interview type using an AI tool.
- User Answer Input: Provide a user interface for the user to input their answers to interview questions.
- Feedback Generation: Generate constructive feedback on user answers, highlighting strengths, weaknesses, and suggestions for improvement, after the simulated interview has been concluded using an AI tool.
- Results Display: Display the interview results, including each question, user's answer, and AI-generated feedback.
- Interview Customization: Allows a user to customize an interview using user provided data

## Style Guidelines:

- Primary color: Deep blue (#293B5F) to convey professionalism and competence.
- Background color: Light gray (#D2D7DF) to ensure comfortable readability.
- Accent color: Soft gold (#A27B5C) to highlight key elements.
- Body font: 'Inter', a grotesque-style sans-serif with a modern look; recommended for body text.
- Headline font: 'Space Grotesk', a proportional sans-serif with a techy feel; suitable for headlines; recommended for headlines.
- Use professional and easily recognizable icons for navigation and interview type selection.
- Clean and structured layout to ensure clarity and easy navigation during the interview process.
- Subtle transitions and animations to enhance user experience without being distracting.
